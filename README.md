# nexui-kit
Low-level UI component library for react projects
## Install

```bash
npm install --save nexui-kit
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'nexui-kit'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
